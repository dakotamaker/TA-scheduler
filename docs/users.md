# Users
Users have a number of different properties, some public,
able to be viewed by any user, and some public, only available to 
supervisors and administrators

## User Properties
- First name (public)
- Last name (public)
- Role (public)
- Office Hours (public)
- Office location (public)
- UWM email (public)
- Password (private)
- Phone number (private)
- Address (private)
